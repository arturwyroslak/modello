# Modello — Landing page portfolio fotomodelki

Prosty, nowoczesny i responsywny landing page dla fotomodelki. Zawiera sekcje: hero, portfolio, about, booking i kontakt.

Jak uruchomić lokalnie:

1. Sklonuj repozytorium
2. Otwórz `index.html` w przeglądarce

Następne kroki:
- Dodać optymalizowane obrazy i lazy-loading
- Integracja z headless CMS (Sanity/Contentful)
- Dodać CI/CD i testy wizualne
